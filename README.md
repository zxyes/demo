# demo
git config --global user.name""   指定用户名
git config --global user.email""  指定邮箱
$ mkdir repository  
$ cd repository  创建仓库
$ pwd 		 显示当前目录
$ git init	 初始化仓库
git添加文件需要add，commit两步
$ git add readme.txt  添加到暂存区
$ git commit -m ""  -m后面输入的是本次提交的说明  commit提交修改

$ git status  显示仓库当前状态
$ git diff  查看修改内容
$ git log命令显示从最近到最远的提交日志
$ git rm <>  删除文件
$ git push origin master
